フルスタックエンジニアとして、採用プラットフォームの開発からERPシステムの導入まで、幅広い領域で活躍してきました。現在は、コーディングテストプラットフォームと採用マッチング事業において、フロントエンド開発を中心にバックエンド開発、外部サービス連携まで担当しています。

技術面では、TypeScript/React によるモダンなフロントエンド開発、Go によるバックエンド開発を行っています。また、Kubernetes や Google Cloud を活用したクラウドネイティブな開発まで、多様な技術スタックでの実務経験を有しています。Monaco Editor を活用したコードレビュー機能の実装や、Google Calendar API との連携による面接自動化機能など、ユーザー体験を重視した機能開発に注力してきました。

過去には、ERPシステムの導入プロジェクトでカスタマイズ開発、システム設計、保守運用を担当し、大規模なエンタープライズシステムの構築経験を積んでいます。これらの経験を通じて、ビジネス要件を技術的に実現する能力と、システム全体を俯瞰して設計する力を培いました。

エンジニアとしての技術力に加え、企業訪問による採用要件のヒアリングやエージェントチームとの連携など、ビジネスサイドとのコミュニケーション能力も強みです。技術的な視点からビジネスに貢献し、チーム全体の生産性と成果物の品質向上に積極的に取り組んでいます。

常に最新の技術トレンドをキャッチアップし、新しい言語や開発手法の習得に意欲的です。競技プログラミングへの参加やRust、Go などの学習を通じて、継続的な自己研鑽に励んでいます。React Tokyo コミュニティの運営メンバーとして、Discord Bot の開発にも携わり、Deno/TypeScript によるスレッド通知機能の実装やコードベースのリファクタリング、自動テスト導入を行いました。業務外でもオープンソース開発やコミュニティ貢献を通じて、技術力の向上とエンジニアリングコミュニティへの還元に努めています。

ユーザーファーストの視点で革新的なソリューションを提供することで、組織の成長と社会への貢献を実現したいと考えています。
